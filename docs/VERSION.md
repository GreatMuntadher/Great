## الأقسام الجديدة

### 1. كروت أساسية (6 موجودة + 6 جديدة)
الجديدة: تحديات نشطة، تحديات مكتملة، متوسط التحديات، متوسط 7 أيام، متوسط 30 يوم، أيام هذا الشهر بيانات.

### 2. ملخص ذكي (`st-insight`)
جمل مبنية على البيانات فقط — اتجاه الأداء، الأيام النظيفة، التحديات النشطة، أكثر محفز، أيام الشهر الجيدة/الصعبة.

### 3. موجة الأداء (`st-wave`)
SVG cubic bezier — آخر 30 يوم — خط متموج ناعم + fill gradient + خط متوسط أفقي + نقاط + animation `stDrawLine`. يعرض empty state عند نقص البيانات.

### 4. تحليل التحديات (`st-challenges-wrap`)
5 كروت إحصاء (نشطة/مكتملة/متوقفة/متوسط/أيام منجزة) + أفضل تحدي تقدماً + الأقرب للاكتمال + progress bars.

### 5. تحليل الإنجاز اليومي (`st-daily-ach`)
6 كروت (متوسط 7 أيام، متوسط 30 يوم، أفضل يوم، أصعب يوم، أيام جيدة، أيام صعبة) + badge اتجاه الأداء (صاعد/ثابت/منخفض) + زر للكالندر.

## طريقة حساب الإحصائيات

```
stGetDayScore(ds) =
  assessments[ds].scores.dailyProgressScore  ← إذا موجود
  أو Math.round(done / TASKS.length * 100)   ← إذا فيه مهام مسجلة
  أو null                                     ← لا بيانات
```

اتجاه الأداء: مقارنة متوسط آخر 7 أيام بمتوسط الـ 7 أيام قبلها:
- فرق > 5% → صاعد 📈
- فرق < -5% → منخفض 📉
- غير ذلك → ثابت ➡️

## طريقة رسم موجة الأداء

SVG خالص — `viewBox="0 0 600 120"` — cubic bezier curves:
```
M x0,y0  C cx,y0 cx,y1 x1,y1  C ...
```
مع fill gradient + خط متوسط أفقي (stroke-dasharray) + animation `stDrawLine` (stroke-dashoffset: 2000→0).

## التحسينات المضافة

### 1. إحصائيات الصفحة (Page Stats)
شبكة 6 كروت أعلى الصفحة تعرض:
- عدد التحديات النشطة / المكتملة / المتوقفة
- متوسط التقدم %
- أطول تحدي نشط (بالأيام)
- مجموع الأيام المنجزة عبر كل التحديات

### 2. شريط الفلترة والترتيب
- **فلترة:** الكل / نشط / مكتمل / متوقف (pill buttons)
- **ترتيب:** الأحدث / الأعلى تقدماً / الأقل تقدماً / الأقرب للاكتمال
- فلترة الواجهة فقط — لا تؤثر على البيانات

### 3. كروت التحديات
- accent border علوي ملون حسب نوع التحدي (بناء=أخضر، ترك=أحمر، عام=أزرق)
- عرض: الاسم + النوع + تاريخ البداية + حالة
- 4 إحصاءات: إجمالي / منجز / تخطي / متبقي
- Progress bar بتدرج + glow effect + رسالة اكتمال

### 4. شبكة الأيام
- اليوم الحالي: glow أزرق + border خاص (`.ch-day-today`)
- للتحديات > 63 يوم: يُظهر أول 63 يوم مع زر "عرض الكل"
- `overflow-x:auto` يمنع كسر الصفحة
- legend يشمل "اليوم الحالي"

### 5. أزرار منظّمة
- **Primary actions:** "أنجزت اليوم" + "تخطيت اليوم" (grid 2 أعمدة)
- **Secondary actions:** إيقاف / استئناف / إعادة ضبط / حذف (compact row)
- حالة disabled واضحة بعد التسجيل

### 6. Empty State محسّن
- Empty State مختلف: بدون تحديات vs لا يوجد في الفلتر الحالي

### 7. Form card
- نُقل للأسفل (يبقى متاحاً عبر زر "تحدي جديد" في الهيدر)
- تصميم أنظف مع icon وعنوان

## طريقة اختبار صفحة التحديات

1. افتح الموقع → اضغط **"التحديات"** في القائمة الجانبية
2. أدخل اسم تحدي → اختر النوع والأيام وتاريخ البداية → اضغط **"إطلاق التحدي"**
3. يظهر الكارد مع شبكة الأيام وشريط التقدم
4. اضغط **"تم إنجاز اليوم"** → يزيد العداد وتتحدث النسبة
5. اضغط مرة ثانية → يظهر إشعار "سبق تسجيل اليوم" دون تغيير
6. اضغط **"إيقاف"** → تتغير حالة الكارد
7. اضغط **"استئناف"** → يعود نشطاً
8. أغلق المتصفح وافتحه → التحديات لا تزال موجودة
9. للتحقق من localStorage: `JSON.parse(localStorage.getItem('recovery_challenges'))`

## الميزات الموجودة حالياً

### اللوحة الرئيسية (Dashboard)
- عداد الأيام النظيفة مع تلوين حسب المرحلة
- بطاقة الحالة النفسية الحالية
- بطاقة تقييم اليوم (آخر نتيجة أو دعوة للبدء)
- بطاقة خطة الحماية اليومية
- بطاقة التقرير الذكي الأسبوعي
- مقتطفات الانتصارات والأهداف

### بروتوكول الطوارئ (SOS)
- 4 مراحل: الإغاثة الفورية / التعزيز / الانتصار / بعد العاصفة
- اختيار سبب الطلب من قائمة جاهزة
- مؤقت تنازلي في المرحلة الأولى
- اختيار شعور من 12 عاطفة
- رسالة انتصار نهائية

### خطة اليوم
- 10 مهام يومية قابلة للتفعيل/التعطيل
- حفظ حالة كل يوم بشكل منفصل في localStorage
- شريط تقدم يومي
- خطط المحفزات المدمجة (9 محفزات + خطة عامة)

### مسار الرحلة (Timeline)
- 5 مراحل: الأيام 1-7، 8-14، 15-30، 31-60، 61-90
- كل مرحلة بطاقة مستقلة مع أيقونة، عنوان، نطاق الأيام، رسالة
- شريط تقدم لكل مرحلة نشطة
- قائمة مهام لكل مرحلة مع تمييز المكتمل والنشط

### تقييم اليوم
- 20 سؤالاً مقسّمة على 6 فئات
- dq_1–dq_10: نعم/لا (حماية، خطر، جسد، تقدم، انتصار)
- dq_11–dq_17: تقييم 1-10 (رغبة، توتر، حزن، وحدة، ملل، نوم، التزام)
- dq_18–dq_20: كتابة قصيرة (انتصار، ثغرة، قرار الغد)
- حساب 5 درجات مستقلة + درجة إجمالية + مستوى خطر
- مدير الأسئلة: تفعيل/تعطيل/إضافة/حذف
- إحصاءات أسبوعية وشهرية

### المحفزات (Triggers)
- 9 محفزات مسبقة التجهيز + إضافة مخصصة
- كل محفز له خطة كاملة: إجراءات فورية، قواعد وقاية، بدائل
- تتبع تاريخ كل محفز

### المذكرات اليومية
- تقييم الجلسة مع درجة خطر
- أسئلة مخصصة قابلة للتعديل
- أرشيف قابل للبحث والفلترة
- تحليل النمط على مدار الوقت

### السجل اليومي
- كتابة حرة مع حفظ تلقائي
- تاريخ كامل لكل يوم
- استيراد تقييم اليوم إلى السجل

### قوانيني
- 6 قواعد افتراضية
- إضافة/حذف/ترتيب القواعد

### أسبابي
- 6 أسباب افتراضية
- إضافة/حذف/تعديل الأسباب

### التقارير الذكية (SWR)
- تقرير أسبوعي تلقائي يُحسب كل أحد
- 6 مقاييس: تقدم، حماية، مزاج، جسد، انتصارات، التزام
- توصية أسبوعية بناءً على الأداء
- أرشيف التقارير السابقة

### المراجعة الأسبوعية
- 4 أسئلة انعكاسية أسبوعية
- حفظ وعرض تاريخ المراجعات

### الإحصاءات
- نظرة عامة على جميع البيانات
- إحصاءات التقييم الأسبوعي والشهري

### وضع الإغلاق (Recovery Lock)
- وضع حماية مؤقت لـ 24 ساعة بعد الانتكاسة
- قائمة مهام الحماية الفورية
- رسائل تحفيزية

### الخصوصية
- الإخفاء السريع (Quick Hide) — يغطي الشاشة فوراً
- قفل PIN من 4 أرقام
- تفعيل/تعطيل كل ميزة مستقلة

### النسخ الاحتياطي
- تصدير كامل البيانات كـ JSON
- استيراد من ملف JSON
